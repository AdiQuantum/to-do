Student Task Manager (Minor Project)
 Overview

Student Task Manager is a simple and interactive web application built using HTML, CSS, and JavaScript.
It helps students manage their daily tasks efficiently without using any backend or database.

All tasks are stored in the browser using LocalStorage, so the data remains even after refreshing the page.

This project is designed as a minor project to demonstrate core front-end concepts such as DOM manipulation, event handling, and browser storage.

Features

Add tasks using button click or Enter key

Automatic date & time added when task is created

Mark tasks as completed

Edit existing tasks

Delete individual tasks

Clear all tasks at once

Search tasks in real time

Task priority (High / Medium / Low)

Keyboard support:

Enter → Add task

↑ Arrow → Focus input box

Dark / Light mode toggle

Floating theme button at bottom-right

Theme preference saved using LocalStorage

Responsive and user-friendly UI

 Technologies Used

HTML – Structure of the application

CSS – Styling and theme management

JavaScript – Application logic and interactivity

LocalStorage – Persistent data storage in browser
